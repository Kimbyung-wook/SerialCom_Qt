# SerialCom_Qt
유명한 프로그램인 SerialCom을 Qt로 만들어보는 중이다.<br>
SerialCom은 임베디드 시스템을 다루는 사람들에게 익숙하고 쉽게 접하는 프로그램이며, Serial program 을 검색하다보면 나오는 프로그램이다.<br>
SerialCom은 NINETY SYSTEM 이라는 국내 회사에서 2000년에 개발되었으며 해당 회사의 주소는 다음과 같다.<br>
